Отчёт по практической работе №5.


Модуль: RetrofitApp (app)

В данном модуле было реализовано приложение, которое получает данные из сети с помощью Retrofit и отображает их в RecyclerView.
Данные загружаются асинхронно, при ошибке выводится сообщение пользователю. Для отображения элементов списка используется отдельный Adapter, а картинки подгружаются через Picasso.

Создан сетевой слой:
RetrofitClient — единая точка создания Retrofit.
ApiService — интерфейс с HTTP-методом получения списка.

Реализован экран со списком:
MainActivity инициализирует RecyclerView и запускает загрузку данных.
После ответа сервера список обновляется через adapter.setData(...).
При ошибке сети показывается Toast.

Реализовано отображение каждого элемента:
TodoAdapter показывает заголовок задачи и статус.
Для картинки используется Picasso.get().load(url).into(imageView).
